# Bug Bounty Simulator

A viral educational browser game about bug bounty triage, severity, scope, safe proof, impact writing, and real fixes.

The game is built as a fast React/Vite app and is designed to be shared on X/Twitter as a score challenge.

## Features

- Russian and English modes
- Junior, Hunter, and Triage Hell difficulty modes
- Realistic Web2, Web3, cloud, API, and blockchain-node bug classes
- Severity selection
- Scope trap decisions
- Safe proof selection
- Fix selection
- Impact argument selection
- Boss rounds against triage archetypes
- Duplicate chance, burnout, reputation, combo, and bounty scoring
- Shareable final result
- Unit tests for scoring logic

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
